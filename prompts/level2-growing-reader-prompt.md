# Prompt: Level 2 Growing Reader Reading Tracker (Grades 2–3)

Use this prompt with any AI assistant (ChatGPT, GitHub Copilot, Claude, etc.) to generate your own customized version of this worksheet.

---

## The Prompt

```
Create a printable reading tracker worksheet for an 8-year-old going into 3rd grade.

Theme: cheerful, colorful, book-themed, but not babyish. Use bright friendly colors, stars, books, pencils, and small coin/penny icons. Make it clean and easy to read.

Title: "Page Pay Reading Tracker"
Subtitle: "Every page I read earns 1 penny!"

Include these sections:

1. Reader Information
- My Name: (blank line for child to write their name)
- Book Title: blank line
- Author: blank line
- Total Pages in This Book: blank line

2. Full Book Earnings
Include a simple formula:
Total pages × 1¢ = money I can earn
Add blanks for the child to fill in:
_____ pages × 1¢ = $_____

3. Daily Reading Log
Create a table with 7 rows and these columns:
- Date
- Page I Started On
- Page I Stopped On
- Pages I Read Today
- Money I Earned Today

Make the table large enough for handwriting.

4. Book Total
Include:
Total pages I read: ______ pages
Total money I earned: $______

5. Money Helper Box
Include:
10 pages = $0.10
25 pages = $0.25
50 pages = $0.50
75 pages = $0.75
100 pages = $1.00

6. Book Reflection
Include:
My favorite part was: ______________________________
I give this book: checkboxes for 1 star through 5 stars.

Design requirements:
- 8.5 x 11 inches, portrait layout
- Output as a single self-contained HTML file
- Use Google Fonts (Fredoka One + Nunito) loaded via @import
- Large clear fonts, plenty of white space
- Include a level badge: "Level 2: Growing Reader · Grades 2–3"
- Encouraging and fun, like a reading rewards program
- Printable in color but still readable in black and white
- Include CSS @media print block for proper 8.5x11 output
```

---

## Customization Tips

- **Add a child's name**: Replace `My Name: (blank line)` with `My Name: [Child's Name]` and style it in a fun color.
- **Add more log rows**: Change "7 rows" to however many days fit your reading goal.
- **Change the reward**: Replace "penny" with "sticker", "point", "minute of screen time", etc.
- **Extend the Money Helper**: Add rows for 150, 200, 250 pages if your child reads longer books.
- **Change the theme**: Swap stars/books for sports, animals, fantasy, etc.

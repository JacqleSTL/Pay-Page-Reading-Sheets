# Prompt: Level 1 Early Reader Reading Sheet (Grades K–1)

Use this prompt with any AI assistant (ChatGPT, GitHub Copilot, Claude, etc.) to generate your own customized version of this worksheet.

---

## The Prompt

```
Create a printable reading worksheet for a 6-year-old going into 1st grade.

Theme: sweet, playful, early-reader friendly. Use rainbows, books, stars, smiley faces, and penny/coin icons. Make it very simple, with large spaces for handwriting.

Title: "Page Pay Reading Sheet"
Subtitle: "Every page I read earns 1 penny!"

Include these sections:

1. Book Information
- My Name: (blank line for child to write their name)
- Book Name: blank line
- How many pages are in the book? _____ pages

2. Today I Read
Use large fill-in blanks:
Today's Date: ________
I started on page: ________
I stopped on page: ________
I read this many pages today: ________ pages

3. My Money
Include simple wording:
Every page = 1 penny!
I read _____ pages.
I earned _____ pennies = $_____

4. Color My Pennies
Add 50 small penny circles that can be colored in.
Arrange them in 5 rows of 10 circles.
Label this section: "Color one penny for each page you read today!"

5. My Reading Feeling
Add checkboxes with simple options:
- Easy
- A little tricky
- Hard
- Fun
- I want to keep going!

6. My Book Rating
Add 5 star rating checkboxes:
1 star, 2 stars, 3 stars, 4 stars, 5 stars

Design requirements:
- 8.5 x 11 inches, portrait layout
- Output as a single self-contained HTML file
- Use Google Fonts (Fredoka One + Nunito) loaded via @import
- Very large readable font, big handwriting lines
- Minimal clutter, friendly and motivating for a first grader
- Include a level badge: "Level 1: Early Reader · Grades K–1"
- Printable in color but still clear in black and white
- Include CSS @media print block for proper 8.5x11 output
```

---

## Customization Tips

- **Add a child's name**: Replace `My Name: (blank line)` with `My Name: [Child's Name]` and style it in a fun color.
- **Change the theme**: Swap out rainbows/stars for dinosaurs, space, princesses, etc.
- **Change the reward**: Replace "penny" with "sticker", "point", or any reward system you use.
- **Adjust grade level**: Mention the grade/age in the prompt to get age-appropriate language.
